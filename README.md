# GONews Bookkeeping Automation
A private automation program for bookkeeping processes connected to Intuit Quickbooks. Handles Invoice and Customer upload via the Quickbooks API, reading and updating Excel sheets, creating customer scannable QR Codes for payable invoice links, and processing docx Mail Merge.


# Privacy Policy
This application is a private internal tool used by Greater Olney News.

The application connects to QuickBooks Online via the official Intuit API.

Financial data accessed is used solely for internal business operations.

No data is sold, shared, or distributed to third parties.

Access is restricted to authorized personnel only.
